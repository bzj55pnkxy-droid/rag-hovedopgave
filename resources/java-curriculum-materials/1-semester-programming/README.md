# Java Curriculum Learning Materials

This directory contains 16 tutorial-style learning materials for the 1st Semester Java Programming curriculum.

## Content Philosophy

- **80% Explanation**: Conceptual understanding, intuition-building, analogies
- **20% Code Examples**: Small, focused, heavily annotated illustrations
- **Audience**: Programming beginners
- **Style**: Patient teacher, not technical manual
